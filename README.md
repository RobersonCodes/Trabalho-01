Trabalho 01 - Git & GitHub
📚 Sobre o Projeto

Este projeto foi desenvolvido como parte da disciplina UC10 - Git & GitHub, com o objetivo de aplicar na prática o controle de versão utilizando o fluxo Gitflow.

🎯 Objetivos do Trabalho
Criar um repositório local
Subir o projeto para o GitHub
Utilizar branches conforme o padrão Gitflow
Trabalhar com versionamento de código
Criar Pull Requests e realizar merges
